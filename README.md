Atividade 1: Astronautas
Contexto
A Agência Espacial Brasileira Digital decidiu organizar melhor suas missões.
Para isso, você deverá criar um sistema em C++ para cadastrar astronautas,
cadastrar voos espaciais e controlar o que acontece com cada missão.
Um voo pode estar apenas planejado, pode estar em curso, pode terminar com
sucesso ou pode terminar em acidente. O sistema deve controlar esses estados e
impedir operações inválidas.
